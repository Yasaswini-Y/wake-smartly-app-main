**Wake Smartly Alarm**
A Smart Alarm Web Application built using React and TypeScript that helps users wake up on time while promoting healthy sleep habits through reminders, weather insights, and motivational content.

**Features**
Create, enable, and delete alarms
Persistent alarms using browser localStorage
Smart alarm checking logic using custom hooks
Pre-sleep reminders
Weather-based information
Motivational quotes for a positive start
Responsive and modern UI

**Tech Stack**
React
TypeScript
Vite
Tailwind CSS
Custom React Hooks

**Project Structure**
src/
├── components/     # Reusable UI components
├── hooks/          # Alarm, weather, and reminder logic
├── lib/            # Utility/helper functions
├── pages/          # Application pages
├── types/          # TypeScript types
├── App.tsx         # Root component
└── main.tsx        # Entry point

**Getting Started**
Install dependencies
npm install

Run the app
npm run dev

Open in browser:
http://localhost:5173

**Key Learnings**
Separation of logic and UI using custom hooks
Modular and scalable React architecture
State persistence using localStorage
Type safety with TypeScript
